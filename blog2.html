What is ProtectedText?

ProtectedText is a free, encrypted notepad to save notes online.It is a free web service where you can write down your texts, encrypt them and access them from anywhere from any device.This website lets the user safely to store all textual content online. It is that simple. All you need is a web browser in an Internet-enabled device.

How does it work?

Each unique notepad has a unique URL, which is further secured by a unique user generated password. Your notes are encrypted using your password, which never leaves your browser. Only the encrypted notes go to ProtectedText’s servers. ProtectedText doesn’t even maintain browser cookies between sessions. This means that even if you accidentally or willingly close your browser while adding notes, your notes remain safe. Sounds pretty amazing, doesn’t it?

Is it safe?

As we are storing our important or any kind of a text, we can get a question that is it safe to trust this site? Well the answer is YES!!! We can trust this web service. The reason is.. ProtectedText site doesn't require any personal information and doesn't store the passwords. There is no ads, no cookies, no user tracking and no registration. No one can view the notes except those who have the password to decrypt the text. Since there is no user registration required, you don't need to create any accounts on this site. Once you done typing the notes, just save and close the web browser and you're good to go!

How it encrypts?

ProtectedText.com uses standard AES algorithm for encrypting/decrypting the content, together with 'salts' and other known good practices to achieve exceptional security; and SHA512 algorithm for hashing. On top of that, all data is only provided through SSL.The encryption is handled by a main.js file which encrypts and decrypts the content locally, so the plaintext should never be stored on the server.

Pros and Cons

The Pros:

  • Your password isn't sent to the server, it can't decrypt your text even if they wanted to. Only encrypted text is sent over the Internet.
  • No logging in or out, just close the browser tab and you're safe.
  • No ads because they can track you, and they're distracting.
  • If someone wants your text, he'll need your password, but he have to find your site first.
  • Access the notes through any device using url and password.
  • You can create many tabs in a particular site.

The Cons:

  • Once you forgot the password there is no way to retrieve the note.
  • If you forget the url your note lost forever.
  • There is no self-destruction feature.
